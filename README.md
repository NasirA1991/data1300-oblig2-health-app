# Hello Health App 2

### Description
CLI Health App built with Python. Allows the user to add health records (name, weight and height), calculates BMI and gives appropriate advice based on BMI results. 
Ability to save health record data to a JSON file, load existing data and display aggregate statistics.


### Installation Instructions

1. Clone the repository: `git clone https://github.com/oslometdata1300/oblig2-health-app-NasirA1991-2.git`
2. Navigate into the folder: `cd oblig2-health-app-NasirA1991-2`
3. Set up project environment: `uv sync`
4. Run the app:
    - Open a terminal/command line inside the project root folder (`oblig2-health-app-NasirA1991-2`)
    - Type: `PYTHONPATH=src python -m health_app.main` to run the program!


### Requirements

* Python 3.14 or higher

* No external dependencies for Oblig 2
